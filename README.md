# Personal-Expense-Tracker-with-Data-Visualization
💸 Personal Expense Tracker with Data Visualization
📌 Project Overview

The Personal Expense Tracker with Data Visualization is a Python-based mini data analytics project that helps users understand their spending habits through automated analysis and visual insights.

It generates synthetic expense data (or can be extended to real CSV inputs), stores it using SQLite, performs financial analysis using Pandas, and visualizes results using Matplotlib.

This project is designed for students and beginners who want to build a strong GitHub portfolio project in Data Analytics and Python development.

🎯 Problem Statement

Most individuals spend money without tracking where it goes. This leads to:

Poor budgeting
Uncontrolled expenses
Lack of financial awareness

This project solves that by:

Categorizing expenses
Tracking income vs spending
Visualizing financial patterns
Generating reports automatically
🧠 Key Features
📊 Automatic expense data generation (or CSV support extension)
🗂 Category-wise expense tracking
💰 Income vs Expense calculation
📅 Monthly spending analysis
📉 Data visualization (bar, pie, line charts)
📁 SQLite database storage
📄 Excel report export
